\doxysubsection{Detailed Description}
This class represents an airport with its ID and name. 

The Airport class encapsulates information about an airport, including its unique three-\/letter airport code (ID) and its name. Objects of this class can be used to store and retrieve airport-\/related information in the Wright\+Flight\+Manager application.
