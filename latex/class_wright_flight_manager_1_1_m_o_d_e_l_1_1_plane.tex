\doxysubsection{Detailed Description}
This class represents a plane with its type and seat capacities. 

The Plane class encapsulates information about a plane, including its type and seat capacities for first class and coach class. Objects of this class can be used to store and retrieve plane-\/related information in the Wright\+Flight\+Manager application.
