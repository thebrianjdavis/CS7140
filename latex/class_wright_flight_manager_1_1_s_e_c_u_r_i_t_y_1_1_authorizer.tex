\doxysubsection{Detailed Description}
This class provides methods for password hashing and verification using SHA-\/512 algorithm. 

The Authorizer class is responsible for generating salted/hashed passwords and verifying if a provided plaintext password matches a stored hash. It uses the SHA-\/512 algorithm for computing the hashes.

For more information about SHA-\/2 and the SHA-\/512 algorithm, please refer to the Wikipedia page\+: \href{https://en.wikipedia.org/wiki/SHA-2}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+SHA-\/2}}
